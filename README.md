## Installation

Install my-tool on termux by using thoes commands 



```bash
pkg update -y && pkg upgarde -y
apt-get update
pkg install python -y
pkg install git -y
git clone https://github.com/rootofhunter/ROH-Bomber.git
cd ROH-Bomber
pip install requests 
python roh.py
```
